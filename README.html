Objetivo da Atividade Mapa:

    Desenvolver um sistema para cadastro de protocolos de uma cooperativa.

  • 1ª tela: Listagem dos clientes cadastrados em grid, disponibilizado o botão: Consultar
  • 2ª tela: Tela para Cadastro de protocolos
  • 3ª tela: Tela para Consulta de protocolos

Recursos Utilizados

  • PHP
  • JQUERY
  • PDO
  • Mysql
  • HTML
  • Bootstrap

Como executar o projeto:

  1. - Copiar a pasta do projeto para a pasta c:\Xampp\htdocs\dashboard
  2. - Copiar a pasta Protocolos para a pasta c:\Xampp\Mysql\data
  3. - Executar o arquivo index.php através do localhost do navegador
